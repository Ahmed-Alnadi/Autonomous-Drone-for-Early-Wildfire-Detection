# Autonomous-Drone-for-Early-Wildfire-Detection
Placed 8th place out of 53 groups in Rutgers Engineering Capstone Competition

· Spearheaded hardware design & integration of a long-endurance UAV capable of 4-hour flight time, outfitted with a
thermal camera, environmental sensors, and real-time AI-driven wildfire detection

· Integrated FLIR Lepton 3.1R, GPS navigation, telemetry communication, and a Jetson Nano running a YOLO-based
neural network model for anomaly detection, enabling fully autonomous patrols with low-latency detection

· Performed field trials to verify sensor accuracy, signal range, and system reliability in real-world outdoor conditions
